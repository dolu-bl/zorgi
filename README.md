## Building

### Windows build requirements

* Qt 5.13.1
* Microsoft Visual C++ Compiler 15.0

### Linux build requirements

* Qt 5.13.1
* gcc 7

### Android build requirements

* Qt 5.13.1 for Android
* jdk 1.8
* Android SDK 26
* Android NDK 20
